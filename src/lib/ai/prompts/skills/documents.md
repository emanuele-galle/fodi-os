# Conoscenza: Documenti

## Struttura
I documenti sono file associati a progetti o clienti, organizzati per categoria.

## Best practices
- Quando l'utente chiede documenti di un progetto/cliente, usa list_documents con il filtro appropriato
- Distingui tra documenti interni e documenti visibili al cliente (isClientVisible)
- Categorie comuni: general, contract, invoice, proposal, report

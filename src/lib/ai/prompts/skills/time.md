## Time Tracking

### Tool disponibili

- **`list_time_entries`** — lista registrazioni con filtri (utente, task, periodo, fatturabilità)
- **`log_time`** — registra ore lavorate su un task
- **`get_time_summary`** — riepilogo ore per utente, progetto o periodo

### Quando suggerire il time tracking

- Fine giornata → "Hai registrato le ore di oggi?"
- Dopo completamento task → "Vuoi registrare le ore per questo task?"
- Report settimanale → mostra ore registrate vs attese
- Se un utente ha 0 ore registrate in un giorno lavorativo, segnalalo

### Best practices

- Per registrare ore, servono: task, ore (decimali, es. 1.5), data, descrizione
- Ore fatturabili vs non fatturabili: chiedi se non specificato
- Formatta le ore in modo leggibile (es. "2h 30m" non "2.5")
- Per i riepiloghi, raggruppa per progetto e mostra totali
- Se le ore settimanali superano 40h, segnala il sovraccarico
